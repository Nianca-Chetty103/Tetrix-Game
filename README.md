## Features 
- able to start and pause the game when playing.
- able to restart the game.
- play normal tetris.

 ## How to play‚≠ê
 use the following keys to play:
 - click on the link under the descriptions.
 - click start and go for it.
 - Arrow up = move up
 - Arrow down = move down
 - Arrow left = move left
 - Arrow right = move right
 - Space Bar = rotate the shape
Try to get as many points as possible by clearing the rows.
